# KeNote,

## 是一款简洁高效的个人记账应用，

## 支持收支管理、统计分析、数据导入导出等功能。

---

在线开始 https://kenote.hellowyq.com
------------------------------------

## 功能

- **收支记账** - 快速记录收入和支出，支持自定义分类和备注
- **数据可视化** - 丰富的图表展示，包括趋势图、饼图、柱状图、热力图等
- **账单筛选** - 按类型、分类、日期筛选账单，支持关键词搜索
- **时间轴视图** - 时间轴和热力图两种展示方式，直观查看账单分布
- **数据导出** - 支持导出账单数据为 Excel 格式，方便备份和分析
- **数据导入** - 支持导入备份数据，数据迁移更便捷
- **自定义分类** - 支持添加和删除自定义分类，满足个人需求
- **内置计算器** - 提供便捷的计算工具，日常算账更轻松
- **暗色模式** - 支持亮色/暗色主题切换
- **响应式设计** - 完美适配桌面端和移动设备
- 最重要的 本工具永远**完全免费完全无广告**!!!

## 数据持久化与稳定性

账单数据存储在浏览器 IndexedDB 中，支持万级数据量秒开。设置和自定义分类保存在 localStorage。

首次使用时，旧版本数据会自动迁移到 IndexedDB，迁移完成后自动清理旧数据。

KeNote 专门针对大量数据进行了优化，全力确保多年使用也不会卡顿。

### 性能测试结果

在 **i5-1035G1 + 核显** 笔记本上测试，**10,000 条账单数据**：

- 首页加载：秒开，所有图表动画流畅
- 账单页初始加载：几乎秒开，约1秒左右
- 筛选/搜索：即时响应
- 导出 Excel：10,000 条数据约需 2-3 秒

### 自行测试数据稳定性

1. 生成测试数据：

```bash
python make-test.py 10000
```

2. 导入测试数据：
   
   - 打开 KeNote 应用
   - 进入控制台 → 设备同步
   - 点击「合并同步数据」选择生成的 JSON 文件

3. 测试项目：

- 首页：观察图表加载速度和动画
- 账单页：滚动列表，测试筛选和搜索
- 导出 Excel：测试大数据导出
- 编辑/删除：测试单条操作响应
- 

4.如果测试完毕,想正式使用软件可从账户菜单清空数据后开始记账吧~

### 架构优化

| 优化点             | 方案                                                      |
| ------------------ | --------------------------------------------------------- |
| **存储**     | IndexedDB 替代 localStorage，支持万级数据                 |
| **查询**     | 索引优化 (by-date, by-type, by-category)，O(log n) 复杂度 |
| **分页**     | 首屏仅加载 50 条，滚动自动加载更多                        |
| **筛选**     | 在 IndexedDB 层面过滤，不加载无关数据                     |
| **首页图表** | 使用 `getAllFromIndex` 批量读取，延迟渲染触发动画       |

## 技术

## **框架**: React 19 + TypeScript

- **构建工具**: Vite 6
- **UI 组件**: shadcn/ui + Radix UI
- **样式**: Tailwind CSS
- **路由**: React Router
- **状态管理**: Zustand (支持数据持久化)
- **数据存储**: IndexedDB (账单) + localStorage (设置)
- **图标**: Lucide
- **图表**: Recharts
- **日期处理**: date-fns + react-day-picker

### 环境要求

## Releases(发行版本)

提供预编译版本：

- Web版 (1MB)
- Windows版 (3MB) - 需要 .NET 10 运行时，仅支持 Windows 10 及以上版本
- 安卓版 (4MB)

客户端体积小巧是因为依赖 WebView 进行渲染。

但记得请留至少30MB+空间用于存储您的数据(实测 存储1年的数据约占用10MB),

而部分Windows系统没有自带.net 10的用户需要额外占用100MB空间用于系统自动下载.net 10

安卓版本理论可直接允许,但要支持webview(例如某些特别设备例如手环等可能没有)

还有，本项目在编译后支持部署到Cloudflare Page、Vercel等静态页面托管网站,快速搭建上线网站端

(例如 编译后的静态的网站：https://kenote.pages/dev 和 https://kenote.hellowyq.com)
